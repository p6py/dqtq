Данный проект представляет собой систему автоматизированного анализа конкурсных списков, разработанную для оптимизации работы приёмных комиссий высших учебных заведений. В рамках приёмной кампании комиссии сталкиваются с необходимостью обработки тысяч заявлений абитуриентов, каждый из которых имеет различные баллы и приоритеты выбора образовательных программ.  

Целью проекта является создание удобного и эффективного программного решения, которое автоматизирует эти процессы. Система позволяет загружать и обновлять конкурсные списки из внешних источников, анализировать данные в реальном времени, рассчитывать проходные баллы с учётом приоритетов абитуриентов, визуализировать результаты и формировать подробные отчёты.

Ключевым компонентом системы является уникальный алгоритм расчёта проходного балла. Он не просто сортирует абитуриентов по сумме баллов, но и интеллектуально учитывает приоритетность выбора программ каждым поступающим. Это позволяет точно моделировать конкурсную ситуацию, определять, кто будет зачислен на каждую программу, и отслеживать, как изменяется ситуация день ото дня.

Разработанное веб-приложение построено на языке Python с использованием фреймворка NiceGUI для создания интуитивного пользовательского интерфейса без необходимости написания кода на HTML или JavaScript. Для работы с табличными данными используется библиотека Pandas, графики строятся с помощью Matplotlib, а профессиональные отчёты в формате PDF генерируются библиотекой FPDF. Все данные хранятся в надежной базе SQLAlchemy.

В результате создан готовый к использованию инструмент, который значительно снижает нагрузку на сотрудников приёмной комиссии, минимизирует человеческий фактор, ускоряет процесс анализа и предоставляет абитуриентам и администрации вуза актуальную и наглядную информацию о ходе приёмной кампании

# Инструкция по установка
### 1. Скопируйте репозиторий
```bash
  git clone https://github.com/p6py/dqtq.git
```
### 2. Установите зависимости
```bash
  pip install -r requirements.txt
```

# Запуск основной программы
```bash
  python app.py
```
